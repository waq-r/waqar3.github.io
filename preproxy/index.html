There are three major preproxy resources available online, these and other top resources either belong to github preproxy projects and pages or to proxy sites, interestingly there are some totally irrelevant resources as domain stats website and curl CURLOPT_PRE_PROXY information resorces, here CURLOPT_PRE_PROXY can be given some bonus points as users do look for curl options and one of such option happens to be about proxy setting with name of preproxy. Anyway as you can see most of preproxy web resources on internet, here are top three proxy and github preproxy resources.

GitHub - lathiat/preproxy: pre-exec proxy determination for bash

Lathiat's preproxy script let you set up proxy variables of shell, these proxy varialbles gets set automatically. Default configuration automatically determines to use proxy or not. Configuration example is provided on github repository along with credits to Ryan Caloras for bash-preexe.sh shell script.

CURLOPT_PRE_PROXY

CURLOPT_PRE_PROXY is explained in detail here, user can check socks, ipv4, ipv6, CURLPROXY_SOCKS5_HOSTNAME and other preproxy related shell setting. It can be used as manual to set up proxy on ubuntu/linux distributions.
